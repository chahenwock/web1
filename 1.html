HTML describes the structure of a web page semantically and originally included cues for the appearance of the documents.

HTML elements are th building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be emneded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
